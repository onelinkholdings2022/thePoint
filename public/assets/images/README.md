## Ghi chú

- Định dạng khuyến nghị: `.png` / `.webp` cho ảnh, `.png` cho logo
- Kích thước: tối thiểu 1920×1080 cho ảnh hero, 956×860 cho ảnh section
- Ảnh từ Google Drive: dùng direct download link hoặc kéo về local rồi đặt vào folder này
